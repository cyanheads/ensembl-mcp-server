# 0.1.3 — 2026-06-06

## Fixed

- **`ensembl_query_region`** — Ensembl's `"No slice found for location"` error now maps to `ctx.fail('invalid_region', …)` (code -32007) instead of falling through as a generic NotFound with `data.reason: null`. Resolves [#1](https://github.com/cyanheads/ensembl-mcp-server/issues/1).
- **`ensembl_query_region` empty-result notice** — expanded from a vague intergenic note to include the chr-prefix convention (`13`, not `chr13`) and a pointer to `ensembl_lookup_gene` for coordinate lookup. Notice text now surfaces in both `structuredContent.notice` and `content[]`. Resolves [#2](https://github.com/cyanheads/ensembl-mcp-server/issues/2).
